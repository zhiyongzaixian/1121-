## jQuery
### 1. 特点
1. jQuery是一个js函数库
2. 专门用于操作DOM的js函数库


### 2. 语法特点
1. 读写二合一
2. 隐式迭代
3. 链式调用
4. 完善的ajax封装


### 3. 核心思想 $ || jQuery
1. 作为核心对象使用：属性
    - $.ajax()
    - $.get()
    - $.each()
    - .....
2. 作为核心函数调用: 参数
    - 选择器字符串
    - function(){}
    - DOM code
    - html字符串

## 事件委托
1. 什么是事件委托
    - 将子元素的事件委托在父元素身上
2. 事件委托的好处
    - 减少绑定的次数
    - 新添加的子元素也可以享用同类型事件
    - 提高性能
3. 事件委托的原理
    - 冒泡
4. 事件委托中触发事件的是谁
    - 子元素
5. 如何找到触发事件的元素
    - event.target
